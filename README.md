## Quick Setup

### Where are the settings?

Click the **profile picture in the top-right corner**.

From there, you can:

- Change the website theme
- Enable or disable 18+ content
- Change the title language
- Add and select watch websites

### How to add a watch website

1. Click the profile picture.
2. Open **Watch Website Settings**.
3. Enter the website name.
4. Enter an address beginning with `https://`.
5. Click **Save Address**.
6. Select the website using the radio button.

You can save up to **three websites**.
